Tutorial Membuat Github Page

Disini saya akan menjelaskan langkah - langkah pembuatan github page menggunakan sistem git.

Langkah 1

Login Github menggunakan akun github anda, kemudian buatlah sebuah repositori dengan akhiran .io

Langkah 2

Setelah itu, lakukan commit untuk file HTML, CSS, dan file yang diperlukan. Kemudian, Melakukan push file yang sudah di-commit ke dalam repositori GitHub dan cek status dengan menggunakan Git

Langkah 3

Kemudian, cek kembali repositori di Github. Setelah itu, setting bagian GithubPages dan tunggu beberapa menit maka link sudah bisa diakses

Anomaly Detection

Memodelkan Anomaly Detection menggunakan data sensor pembangkit listrik

(Gambar diatas hanya Ilustrasi) Projek ini merupakan pekerjaan saya saat menjadi Data Science Intern di PT.SML, Data diambil sendiri dari aplikasi salah satu pembangkit listrik yang hanya bisa diambil sebanyak 7 sensor sekali tarik data. Data yang diambil sangat kotor dan acak, kemudian melakukan grouping per equipment agar saat ditrain menghasilkan output yang diinginkan (manipulasi data menggunakan pandas). Setelah itu memodelkan Anomaly Detection menggunakan CNN Autoencoder dan memprediksi sensor-sensor yang terdapat anomali.

  • Name: Anomaly Detection
  • Category: Data Science
